# JSON vs XML vs Binary Encoding

## question
What is a key advantage of binary encoding formats over textual formats like JSON and XML?

## options
- A) Better human readability
- B) Easier debugging with standard text editors
- C) More compact representation and faster parsing
- D) Better compatibility across programming languages

## answer
C

## explanation
Binary encoding formats are more compact (smaller size) and faster to parse compared to textual formats like JSON and XML. While textual formats are human-readable and easier to debug, binary formats trade readability for efficiency in storage and transmission.
